# Smart Healthcare Management System

## Project Overview
This project is a Smart Healthcare Management System developed using Python and Machine Learning.  
It helps manage patient records, appointments, billing, and provides intelligent health recommendations.

## Features
- Patient record management
- Appointment scheduling
- AI-based disease recommendation
- Billing system

## Technologies Used
- Python
- Pandas
- Machine Learning
- Google Colab
